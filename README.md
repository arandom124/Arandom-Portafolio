## Despliegue

Para implementar mi cartera (o la suya propia), ejecute:

```bash
  npm inicio de ejecución
```

De lo contrario, cuando estés trabajando en ello, usa
```bash
  npm ejecutar dev
```
, o:
```bash
  npx próximo desarrollo
```
